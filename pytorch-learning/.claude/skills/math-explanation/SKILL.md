# 数学・行列演算 解説スキル

数学・行列演算の解説依頼を受けたら、このスキルの手順で成果物を作る。

## 手順

1. **必ず** `references/explanation-rules.md` を読む（解説の共通ルール・チェックリスト）
2. 成果物の形式に応じて追加で読む：

| 成果物 | 読むファイル |
|--------|-------------|
| marimo ノートブック（インタラクティブ可視化） | `references/marimo-notebook.md` |
| Markdown 解説ドキュメント | `references/markdown-doc.md` |

形式の指定がない場合は marimo ノートブックを推奨し、ユーザーに確認する。

## プロジェクト設定（他プロジェクトへ移植時はここだけ書き換える）

| 項目 | このプロジェクトでの値 |
|------|----------------------|
| コード例のフレームワーク | PyTorch |
| ノートブック配置先 | `src/basic/<トピック>/` |
| Markdown 配置先 | 解説対象の `.py` と同じディレクトリ、ファイル名 `math_explanation.md` |
| 日本語フォント | `"Hiragino Sans"`（macOS） |
| 実行環境 | `.venv`（uv 管理）。起動: `.venv/bin/marimo run <file>.py` |

## 共通原則（要約）

- 説明文はすべて **日本語**、数式は LaTeX 記法
- 添字は **0 始まり**（Python の慣習に合わせる）
- 構成順序: ①計算方法 → ②何を表現しているか → ③具体的な計算例（全要素を手計算） → ④視覚化 → ⑤使いどころ

詳細は必ず references を参照すること。
