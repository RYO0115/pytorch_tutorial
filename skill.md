# 数学・行列演算 解説テンプレート

数学概念・行列演算・線形代数の解説を求められた際に従うべきルールとテンプレートです。

---

## 解説の原則

### 言語・表記
- 説明文はすべて **日本語** で書く
- 数式は LaTeX 記法（`$$...$$` / `$...$`）を使う
- 行列・ベクトルの添字は 0 始まりで統一する（Python の慣習に合わせる）
- コードは必ず PyTorch で示す

### 解説の構成順序（必須）
1. **計算方法**（数式 + コード）
2. **何を表現しているか**（直感・現象の意味）
3. **具体的な計算例**（数値を使って全要素を手計算）
4. **視覚化**（marimo + matplotlib グラフ推奨）
5. **使いどころ**（実応用との対応）

### 視覚化のルール
- **marimoノートブック** でインタラクティブに示す（`mo.ui.slider` などを活用）
- グラフタイトル・軸ラベルは日本語で書く
- 日本語フォントは必ず設定する（macOS では `"Hiragino Sans"`）

```python
plt.rcParams["font.family"] = "Hiragino Sans"
plt.rcParams["axes.unicode_minus"] = False
```

- ファイルは `src/basic/<トピック>/` 以下に配置する

## 途中計算の書き方フォーマット

### 行列積
```
C[i,j]:  A[i,0]×B[0,j] + A[i,1]×B[1,j] + ...  =  <各積の展開>  =  <合計>

例）
C[0,0]:  1×2 + 3×4  =  2 + 12  =  14
```

### 外積
```
outer[i,j] = a[i] × b[j]

例）
outer[0,1] = a[0] × b[1] = 2.0 × 1.5 = 3.0
```

### 要素積
```
位置 (i,j):  A[i,j] × B[i,j]  =  <値> × <値>  =  <結果>  →  Z[i,j] = <結果>
```
